Introduction of the ecTA / ecPayroll main screen

Make sure you have completed following session(s)

l   System Login

 

After login to the ecTA / ecPayroll system, the main screen will be shows. Here is the introduction about the main screen.

 

 

 

 

 

 

1.          The Main Screen Toolbar

You may find the main system functions by clicking these toolbar buttons. If you point the mouse cursor upon the button, the system will shows the description of the corresponding button. (e.g. Access Log, Device Management, Reporting, etc.

 

2.          The Employee Toolbar

If you want to select an employee record(s) with specific condition, you may click these employee filtering functions by this toolbar.

 

3.          The Company Hierarchy

Here shows the Company & Department structure. If you want to change the structure, please click the Company Setup & Department Setup

 

4.          The Duty Group List

Duty Group Duty means pre-defined roster. For example, if staff work on every Monday - Friday, 09:00 - 18:00 and day off on every Saturday & Sunday regularly. We may defined a Duty Group for those group of staffs. Here shows the list of the Duty Group.

 

5.          The Employee List

Here shows the list of the employee records, you may find more employee details by double clicking the employee records.

 

Remark:

        The toolbar buttons may be affected by below factors:

1.      ecTA / ecPayroll system version

2.      System module purchased

3.      The login user right

How to set “auto clock synchronization”

Make sure you have completed following session(s)

l            None

 

Steps

1.          Run “Transaction Download Manager”

 

2.          Go to “Reader Control” and choose a reader

 

3.     Go to “Reader Properties” 

 

4.   Go to “Advance Setting”

 

5.   Press “System” and key in password

 

6.     Enable “Auto Clock Synchronization” and check the Next clock synchronization time

 

 

***Remark: Keep running “Transaction Download Manager” to ensure the system does clock synchronization regularly***

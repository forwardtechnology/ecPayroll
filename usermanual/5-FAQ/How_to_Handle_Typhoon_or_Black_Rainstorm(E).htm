How to Handle Typhoon / Black Rain Leave

 

Make sure you have completed following session(s)

l            Already create Typhoon / Black Rainstorm Leave on ¡§Item Record¡¨

 

In case of Typhoon / Black Rainstorm, normally, employer will require employee back to work within 2 hours after Typhoon / Black Rainstorm is cancelled.

On ecTA / ecPayroll, user have to add leave records to cover their non-working period

 

Steps

1.            Press [Commander] on main screen toolbar

 

2.            Choose [Add Leave Record]

 

3.            Enter the date range

4.            Choose a leave type (e.g. Typhoon / Black Rainstorm )

5.            If Typhoon / Black Rainstorm is last for a day, please choose [Whole day], otherwise, unselect [Whole day] and enter the cut-off time

6.            The system will auto calculate [days value], adjust the [days value] if needed

7.            Enter the leave remark if needed

8.            Choose employee by [EE Filter]

9.            Check if any mistake. If ok, press [Execute] to process

 

10.         Enter [Execute] on the textbook, then press [OK]

 

11.         Please update the attendance result by re-analysis

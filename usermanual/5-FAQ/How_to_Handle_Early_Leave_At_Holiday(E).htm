How to Handle Early Leave at Holiday (e.g. Mid-Autumn Festival, Winter Solstice, etc)

 

Make sure you have completed following session(s)

l            Already create [Pay Leave] in [Leave Type Setup]

 

 

Steps

1.            Press [Commander] on main screen toolbar

 

2.            Choose [Add Leave Record]

 

3.            Enter the [Date Range]

4.            Choose a leave type (e.g. Pay Leave )

5.            Unselect the [Whole Day], then enter the cut-off time

6.            The system will auto calculate [days value], adjust the [days value] if needed

7.            Enter the [Remark] if needed

8.            Choose employee by [EE Filter]

9.            Check if any mistake. If ok, press [Execute] to process

 

10.         Enter [Execute] on the textbook, then press [OK]

 

11.         Please update the attendance result by re-analysis

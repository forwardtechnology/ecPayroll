Export IR56B (Export database for IRD software)

Make sure you have completed following session(s)

l            Already fill in employee information (e.g. Birth Date, HKID, District, etc)

l            Already have correct salary record

 

***Before export IR56, please make sure all essential items are filled without mistake, otherwise you need to export again after data admendment***

 

Steps

1.          Go to ¡§Payroll Maintenance¡¨

 

 

2.     Go to ¡§Export to IB56B¡¨

 

 

3.      Select ¡§Company¡¨, ¡§Year¡¨ and enter ¡§Date of Submission¡¨

 

 

4.      Choose ¡§Path to Export¡¨. If you want to export to IR56B folder directly, please backup and store the current file in theIR56B\ -> IR56B.mdb first.

 

 

5.      Choose staff who need to export to IR56B in the employee list

 

 

6.      Press ¡§Store and Export¡¨ to export

 

 

7.      When export completed, IR56B file generation report will be generate. You can check if any error while data export

 

 

8.      Copy the export file (ir56b.mdb) to IRD software folder. Default path is C:\IR56B\

 

9.      Run IRD software to check the output. Default password for IRD software is ird

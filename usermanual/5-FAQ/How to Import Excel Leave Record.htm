How to Import Excel Leave Record

Make sure you have completed following session(s)

l            Already setup the employee record

l            Already set up the leave type

 

Steps

1.          Open TAMS\TEMPLATE\ Import leave records v2.xls

 

2.            Enter the leave information according to the reminder

 

3.          Save as new Excel file

 

4.          Go to “File” -> “Data Import”

 

5.          Choose “employee leave record”

 

6.           Choose import path and file name

 

7.          Press “Start Import”

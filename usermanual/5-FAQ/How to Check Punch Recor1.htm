How to Check Punch Record / How to Manual Import Punch Record

Make sure you have completed following session(s)

l           Already download punch record

 

Steps

1.          Choose TAMS32 shortcut, right click and choose “Content”

 

2.          Press “Search Target(搜尋目標)”

 

3.     Choose “Detail” and “Edit date” to view the files

 

4.          Default file Trans.dat############## is downloaded punch record

(############## = Download Year, Month, Day, Hour, Minute, Second)

 

5.     If you want to view the actual Trans.dat file name, please go to “Transaction Download Manager” -> Reader Control -> go to individual Reader Properties and view Transaction File

 

6.      If you want to see records of Trans.dat##############，Please open the file with notepad

 

(Trans.dat format will be different according different punch card / fingerprint / hand key machine versions)

***If you want to get the virtual punch record, the corresponding format needed to follow. (The punch record may not be able to be virtually because it may be encryption)And then import attendance result. *** (We do not suggest you to do the virtual punch record.)

 

7.      If Trans.dat##############‘s record number is different from the TA system’s, you can try to import that trans.dat again##############

 

Manual import Trans.dat  

1.          Go to “Transaction Download Manager” and press “Reader Control”

 

 

2.          Choose punch record (Firstly, choose card reader B. Then import / virtual it’s record)

 

3.     Press “Manual import data” and choose “All” in the file type

 

4.     Search the file which needs to be imported, “Download and import” record will be displayed after imported

 

(Same time, place, PIN’s punch record will not be marked duplicity.)

 

5.      Please re-analysis if you want to get the latest attendance result

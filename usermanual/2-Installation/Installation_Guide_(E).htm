Installation Guide

 

System Overview

 

ecTA / ecPayroll system is designed for working on a Local Area Network (LAN). Main program files and master database files are all installed at SERVER side while users operate the system at WORKSTATION side.

Microsoft SQLServer is required to be installed in the SERVER as the database management system (DBMS) to store all data in the system. On the other hand, some basic library files are required to be installed in all WORKSTATIONs which may run the system.

System installation

Please follow below steps to install the system:

1          Installation at Server

1.1          Install database files

1.1.1        Find below 2 files in the provided CD

l   TAMS.mdf

l   TAMS_SYSTEM.mdf

1.1.2        Attach above 2 database files to the MSSQL Server

l   For details please refer to (Attach MSSQL Database)

1.2          Install ecPayroll Software

1.4          Share the program folder (i.e. C:\Program Files\TAMS\) named as ¡§TAMS¡¨ with FULL control right to the workstation(s) which is going to connect to the system

Example - Fixed Working Hr

 

Example 1

Duty Start Time & End Time

09:00 – 20:00 (Staff have to work 8 Hr within the allowed working period)

Lunch Period

1 Hour (Flexible, No punch card is required)

Overtime

No

Overnight

No

 

 

 

Steps

1.            Go to [Calendar Setup]

 

2.            Go to [Duty Setup]

 

3.            Press [New] to add duty

 

4.            Input Flexible 9-20(8) on the [Description]

 

5.            Select [Fix Hour Work Duty]

 

6.            Input 09:00 – 20:00

 

7.            Input 9 Hr (Including 1 Hr Lunch)

 

8.            Press [Save] to complete setting

Example - Split Shift

 

Example 1

Duty Start Time & End Time

09:00 – 18:00 Fixed

Lunch Period

13:00 – 14:00 punch card required

Overtime

No

Overnight

No

 

 

 

Steps

1.            Go to [Calendar Setup]

 

2.            Go to [Duty Setup]

 

3.            Press [New] to add duty

 

4.            Input 9-13,14-18 on the [Description]

 

5.            Select [Work Duty]

 

6.            Input 09:00 – 13:00

 

7.            Select [Work Duty] Again

 

8.            Input 14:00 – 18:00

 

9.            Press [Save] to complete setting

Employee Record – Basic Salary

Make sure you have completed following session(s)

l   Add New Employee

 

 

“Basic salary” is the amount paid to an employee before any extras are added or deducted, such as reductions because of late deduction or an increase due to overtime or a bonus. Normally, you have to input the Basic salary for each employee before the actual salary can be calculated.

 

Please follow below steps to input Basic Salary for an employee:

 

Steps

 

1          In main screen, select and open an “Employee Record”

 

 

2          Select “Salary Setup” page from the menu on the left.

 

 

3          In the frame of “Current Basic Salary”, you can see the current information of basic salary as at today. If the boxes are blank that mean no any basic salary record found.

 

 

4          Click the [Edit] button to change Basic salary record for this employee.

 

 

5          A Basic Salary Setup window will be shown. On the top, all Basic salary records of this employee will be shown including all history, current, and even some records that are not effective yet.

 

 

6          To add a new Basic Salary record, click the [Add New] button.

 

 

7          Fill in all data fields depends on which scenario :

7.1         Scenario 1: Add the 1st Basic Salary record for a new join employee.

 

Data field

Description

Example

 

Staff A

Join date : 2015/3/25

Employment status : Active

Salary pay cycle : Monthly

Effective from

The effective start date of this Basic Salary entry.

 

The value should be the join date by default.

 

Basic salary effective from : 2015/03/25

 

Effective to

The effective end date of this Basic Salary record.

 

Normally, it should be “No end date” as this Basic Salary is effective without a specified end date at this moment.

 

Attention : Although you can set an effective end date, you are not recommend to do it or otherwise the actual salary may wrongly calculated if there is no any basic salary are effective with the calculated salary period.

 

Basic salary effective to : No end date

Salary Type

The type of Basic salary.

 

There are 4 options: Monthly / Weekly / Daily / Job. If none of the above are suitable, please select “Monthly”

Salary Type : Monthly

Basic Salary

The amount of the Basic salary.

 

This value must not be negative.

Basic salary amount : 10,000

        

        

        

7.2         Scenario 2 : Change Basic Salary for an employee

 

Data field

Description

Example

 

Last Basic Salary record

Effective from date : 2015/03/25

Effective end date : “No end date”

Amount : $10,000

 

Effective from

The effective start date of this Basic Salary entry.

 

The date can be a future date when you want to pre-set the basic salary and take effect in future, say next month. Before the effective date, when you calculate salary, the current basic salary as of today will be used.

 

This date must be later than the effective start date of the last Basic salary record, and the effective end date if it is not “No end date”

 

If your entered effective date is as same as your salary period start date, the exact amount of your defined Basic Salary will be applied to the formula for salary calculation. If not, the Basic Salary will be calculated in prorate automatically.

 

Example 1 :

New Basic Salary Effective from date : 2018/05/01

 

Example 2 : Enter a future effective date

New Basic Salary Effective from date : 2020/01/01

 

Example 3 : Effective date not same as the salary period start date

Salary period : 2018/06/01 – 2018/06/30

 

New Basic Salary Effective from date : 2018/06/21

New Basic Salary Amount : $15,000

 

Pro rata Basic Salary = $10,000 x 20/30 + $15,000 x 10/30 = $11,666.67

 

Effective to

The effective end date of this Basic Salary record.

 

Normally, it should be “No end date” as this Basic Salary is effective without a specified end date at this moment.

 

Attention : Although you can set an effective end date, you are not recommend to do it or otherwise the actual salary may wrongly calculated if there is no any basic salary are effective with the calculated salary period.

 

Basic salary effective to : No end date

Salary Type

The type of Basic salary.

 

There are 4 options: Monthly / Weekly / Daily / Job. If none of the above are suitable, please select “Monthly”

Salary Type : Monthly

Basic Salary

The amount of the Basic salary.

 

This value must not be negative.

Basic salary amount : 15,000

               

               

 

8          Click the [Update] button to save changes.

 

 

9          Click the [Close] button to return to the employee record

 

 

10      Finish

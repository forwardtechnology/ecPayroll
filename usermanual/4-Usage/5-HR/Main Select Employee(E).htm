Steps

 

Method 1: By select Department Tree

 

1.1  Click the department from the main screen department tree.

 

 

1.2  The employee record(s) that under the selected department will be shown on the main screen.

 

 

 

Method 2: By select duty group

 

2.1  Click the duty group from the main screen duty group list.

 

 

2.2  The employee record(s) that under the selected duty group will be shown on the main screen.

 

 

 

Method 3: By search employee ID

 

3.1  Enter the employee ID of the employee from the main screen search ID field, then press the [Enter] key

 

 

3.2  The employee record with the entered employee ID will be shown on the main screen.

 

 

Method 4: By search employee name

 

4.1  Click the [Search ID] button, then it is turn to [Search Name]

 

 

4.2  Enter the employee name (Surname, First Name, Chinese Name, Nickname are both accepted.) of the employee, then press the [Enter] key

 

 

4.3  The employee record with the entered characters will be shown on the main screen.

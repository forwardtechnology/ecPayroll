View / Edit Employee Record

Make sure you have completed following session(s)

l   Add New Employee

 

To View / edit an employee record, please:

 

Steps

 

1.      Find out the employee record that you are going to view / edit (User Manual: Find out an Employee Record )

 

2.      Double the employee record from the main screen (e.g. Employee ID: A0100 )

 

 

3.      Select the corresponding page that you are going to view / edit.

 

 

4.      To edit the employee record, enter the information, then click the [Update] button to save the changes.

 

 

 

NOTE

The [Update] button most likely appear of the top left corner of the employment record. However, for some pages, like Employment, Movement, Education, Work Exp., Leave Record, etc. As those pages are designed for user to enter multiple records, therefore, the [Update] button nearby the list table.

In ecPayroll system, user can input MPF plan details for each employee. Please see below steps:

 

Steps

1.          Open the screen ¡§Employee Record¡¨ of the employee to be changed.

 

2.          Click on the menu item ¡§MPF Plan¡¨ on the left hand side.

 

 

3.          Fill in each data field accordingly. See below a brief description of each field.

 

Field

Options

Description

Provident fund type

NA / MPF / ORSO

N/A : Select N/A if the employee has no any provident fund plan.

MPF : If employee join MPF plan

ORSO : If employee join ORSO plan

MPF Provider

The MPF Provider(s) selected in Company Setup

If there are 2 MPF providers were set in Company setup, select the employee preferred one here.

If only 1 MPF provider were set in Company setup, that MPF provider were selected as default here.

 

This option is one of the factor affecting which employee(s) will be exported in the process of MPF data export.

 

For more information, click here

Account number

 

The account number of the MPF / ORSO account

MPF Scheme > Staff Type

Regular Employees /

Casual Employees

This option affect the calculation of MPF contribution.

MPF Scheme > Scheme Type

Master Trust Scheme /

Industry scheme

Industry scheme is specially established for employees of the catering and construction industries, particularly casual employees.

 

This option affect the calculation of MPF contribution.

Individual Voluntary Contributions > Employee VC %

 

This value take effect ONLY if you have enabled voluntary contribution and selected VC% in salary profile. This % value is used for calculating Employee Voluntary Contribution amount.

Individual Voluntary Contributions > Employer VC %

 

This value take effect ONLY if you have enabled voluntary contribution and selected VC% in salary profile. This % value is used for calculating Employer Voluntary Contribution amount.

Plan details > Benefit Plan No.

 

The benefit number of the joined MPF/ORSO plan. This value may be shown in MPF report and MPF data export file.

Plan details > Plan Join Date

 

The join date of MPF / ORSO plan. This value will be shown in MPF report and MPF data export file.

Plan details > Plan Termination Date

 

The termination date of MPF / ORSO plan. This value will be shown in MPF report and MPF data export file.

Plan details > Plan Termination Reason

 

The termination reason of MPF / ORSO plan. This value will be shown in MPF report and MPF data export file.

 

 

4.          Press the [Update] button to save the changes.

 

 

5.          Finished.

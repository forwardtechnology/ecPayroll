Print Wages List

Make sure you have completed following session(s)

l            None

 

Steps

1          Click the Report on the main screen, then select the Employee Wage List

 

2          Click Report Filter button

 


 

then choose Filter Item to make selection

  

       

        Salary Profile is select the profile you want to print

After selecting the profile, click OK

 

       

 

        Department is a choice for printing the employee of that department in the payroll period

         (Only print the employee of that department in the payroll period. If the employee have already leaved, it will be also displayed their name.)

         Only select department then click OK

 

3           When you have finished the steps, click the Setting button

 


 

It will be displayed the payroll item for Employee Wages List printing. (For example: The position of item)

Delete a User Template in software

Make sure you have completed following session(s)

l   What is a User Template?

l   Viewing User Template list stored in software

 

 

This topic will walk you through how to delete an existing user template in software database. When you want to cancel the user template due to the corresponding employee is resigned, you may need to use this function to delete the user template, then restore the updated user template list to corresponding Time Clock Device(s) by restore template to device function.

 

 

Steps

1.          View the User Template List by following the instruction of View User Template List stored in software

 

 

2.          Select the user template that you are going to delete (e.g. User ID: 10027)

 

 

3.          Click the [ Delete User ] button from the toolbar

 

 

4.          Click [ Yes ] to confirm delete

 

 

 

 

CAUTION

Please check carefully before confirmation due to the deleted user template cannot be retrieved again.

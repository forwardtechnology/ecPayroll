View User Template List stored in software

Make sure you have completed following session(s)

l   What is a User Template?

 

 

This topic will walk you through how to view User Template List in the software. There are some situations we may need to view User Template List in the software:

       

1.      To check the User Template(s) after backup User Templates from Device

 

2.      To confirm the User Template(s) are ready before restore User Templates to one or more device(s)

 

 

Steps

1.          Click the [ Device Management ] button from the main screen toolbar

 

 

2.           Click the [ User Edit ] button from the toolbar

 

 

3.          Select the device model to view the user template list. Time Clock model that support this function are listed as below:

 

Supported Time Clock Device Model:

TA204 / TA205 / TA206 Smart Card Terminal

TA207 Smart Card / Fingerprint Terminal

ACTAtek Smart Card / Fingerprint Terminal

F4 / F702 Fingerprint Terminal

 

 

 

4.          You may use the sorting function to view the user template list by clicking the header of the user template list.

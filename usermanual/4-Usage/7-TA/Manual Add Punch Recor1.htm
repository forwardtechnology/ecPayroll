Manual Add Punch Record

Make sure you have completed following session(s)

l            Already setup employee

l            Already assign PIN card

 

Steps

1.          Main Screen -> Choose the employee -> Open Timetable

 

2.     Choose “Punch Record” and “Date Range” -> Display

 

3.     Go to “Punch Card Record” -> “New” and add time

 

 ***Attention: If the punch record needs to count for overnight, need to choose the next day punch card record****

***If you want to view the latest attendant result, please re-analysis that day***

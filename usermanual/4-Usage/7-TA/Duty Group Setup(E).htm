Duty Group Setup

Make sure you have completed following session(s)

l            None

 

Steps

1.       Select Duty Group Setup in Setup menu or click Department Setup icon on toolbar.

 

2.       Duty Setup Window is displayed.

 

3.         Adding New Duty Group

3.1.  Click the New icon on the toolbar

 

3.1.  Enter Description in description text box.

 

3.2.  Click the Update icon to save

 

4.         Editing Duty Group

4.1.  Select a proper duty group item by clicking navigation icons on the toolbar.

 

4.2.  You can follow the instruction shown in the pervious section

 

5.         Deleting Duty Group

5.1.  Select a proper duty group item by clicking navigation icons on the toolbar.

 

5.2.  Select Delete icon

 

5.3.  Click Yes to confirm

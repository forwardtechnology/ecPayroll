Backup User Templates from Device(s)

Make sure you have completed following session(s)

l   What is a User Template?

 

 

Backup User Template from Device means transferring all of the User Template(s) from the selected Time Clock Device to the software database.

 

When one or more user templates are directly added / changed / deleted on the Time Clock Device, you are suggested to backup the updated user template list to the software database for safety and further transfer purpose.

 

 

 

TIPS

If you have multi devices installed. You are suggested to use only one particular device as a master enrollment unit to do all user enrollments. So that you can manage user templates easily by simply transfer all user templates from master unit to all other devices.

 

 

Steps

1.          Click the [ Device Management ] button from the main screen toolbar

 

 

 

2.          Select the device to be backup (e.g. Site8). If user enrolments are done on several devices, you may multi-select them to perform backup

 

 

3.          Click the [ Backup Template ] button to start backup

 

 

CAUTION

DO NOT shut down the software or the device(s) until the backup process completed.

 

 

4.          The backup result will be shown on the communication log after process completed.

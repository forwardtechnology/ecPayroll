Update a User Template in software

Make sure you have completed following session(s)

l   What is a User Template?

l   Viewing User Template list stored in software

 

 

 

 

 

NOTE

 

This topic is only suitable for below Time Clock Device model:

 

TA204 / TA205 / TA206 Smart Card Terminal

TA207 Smart Card / Fingerprint Terminal

ACTAtek Smart Card / Fingerprint Terminal

F4 / F702 Fingerprint Terminal

 

 

This topic will walk you through how to update an existing user template in software database. When you want to assign a new card no / password for an existing employee (Due to Lost card / forgot password), you may need to use this function to update the user template, then restore the updated user template list to corresponding Time Clock Device(s) by restore template to device function.

 

 

Steps

1.          View the User Template List by following the instruction of View User Template List stored in software

 

 

2.          Select the user template that you are going to edit, e.g. User ID: 10027

 

 

3.          Edit the user template at the bottom of the setting screen. E.g. Change the password from 10027 to 9999

 

 

4.          Click the [ Update ] button to save the user template. After all, you will able to check the updated result on the user template list.

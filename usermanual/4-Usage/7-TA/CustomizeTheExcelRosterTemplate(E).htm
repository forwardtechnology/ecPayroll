Customize the Excel roster template

Make sure you have completed following session(s)

l   None

 

 

Instead of using our provided duty roster EXCEL template, you may customize it in your own format to fit your business in the real world.

 

However, of course, there are some limitations or otherwise it fails to be imported to the system. Basically the system locates the required data blocks by checking the cell Tag (i.e. the name of a single cell, or a block of cells) instead of checking the coordinate in column and row (e.g. A4). Therefore, all existing Tags in the Excel template should NOT be changed or deleted. In the provided Excel template, the related data block are highlighted in yellow for easy reference.

 

 

 

NOTE

You may find the build-in roster Excel template in below folder :

 

\TAMS\template\Import\RosterImport(Template).xlt

 

 

Generally, below format you may consider to change if required:

 

1.          Filename of the template

2.          Font style and size

3.          Cell forecolor / background color

4.          Title in the template

5.          Fixed text remarks or instruction

6.          Insert more columns for staff information

7.          Insert more rows for day information

8.          Insert some Excel formula to do some counting or calculations.

 

 

 

CAUTION

Please backup the original template file before edit your own.

 

 

Below is an example for reference:

 

Steps

1.          Find the build-in roster Excel template in below folder:

\TAMS\template\Import\RosterImport(Template).xlt

 

 

 

2.          Open the Excel template by mouse right-click and then select “Open”. Do not open the file by mouse double-click or otherwise you cannot edit the template.

 

 

3.          Edit the Excel template to fit your business.

 

E.g.1 Insert some Excel formula to do some counting or calculations.

 

       

        E.g.2 Insert some fixed text remarks or instruction

               

               

 

4.          When edit finished, save the Excel template

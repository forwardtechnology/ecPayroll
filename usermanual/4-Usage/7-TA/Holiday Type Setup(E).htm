Holiday Type Setup

Make sure you have completed following session(s) 

l            None

 

Steps

1          Click the Item Record icon on the main screen toolbar

 

2          Select the Holiday Type Setup

 

3          It will show the setting screen

 

4          Adding new holiday type

4.1         Click the New icon on the toolbar

  

4.2         Type in Holiday Type Name

4.3         Click the Update icon on the toolbar to save

 

5          Editing Holiday Type

5.1         Select the Holiday Type you want to edit

5.2         You can follow the instruction shown in the pervious section

 

6          Deleting Holiday Type

6.1         Select the Holiday type you want to delete

6.2         Click the Delete icon on the toolbar

6.3         Click Yes to confirm

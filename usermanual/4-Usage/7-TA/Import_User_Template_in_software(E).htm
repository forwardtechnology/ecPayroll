Import User Template in software

Make sure you have completed following session(s)

l   What is a User Template?

l   Viewing User Template list stored in software

l   Add a User Template in software

 

 

 

 

 

 

NOTE

 

 

This topic is only suitable for below Time Clock Device model:

 

TA204 / TA205 / TA206 Smart Card Terminal

ACTAtek Smart Card / Fingerprint Terminal

 

 

 

Instead of adding new user template one by one, you can batch import the user template list from a data file. This function is useful especially when the system is new installed and you want to migrate the user list (i.e. permitted card list) from old system.

 

Please follow the steps below:

 

Steps

1.          Prepare the data file to be imported. Depends on different device model, the file format is different. See below table:

 

Model

File format

Fields

Example

Remarks

TA204 / TA205 / TA206

Text, CSV

1st field: User ID

(Max. 5 Digit)

 

2nd field: Card number

(Max. 10 Digit)

 

3rd field : Password  (Max. 5 Digit)

 

4th field : Privilege

     1 = User

     3 = Admin

“159","12345","321","1"

“160","11122","147","3"

“170","23423","258","1"

 

 

In case the User number already exist in the device, the Card number, Password, and the Privilege will be updated.

ACTAtek

Text, CSV

1st field : User ID

2nd field : Card number

3rd field : Control flag

     0 = Add / Update

     1 = Delete

"01100419","A30B05E5","0"

"01100449","532C02E5","1"

"01005799","835106E5","0"

 

 

 

2.          Open the User Template List screen by following the instruction of View User Template List stored in software .

 

 

3.          Select your device model. (e.g. TA206 )

 

 

4.          Click the [ Import ] button from the toolbar

 

 

5.          Browse and select your prepared data file, then click the [Open] button to start import file.

 

 

6.          After file import, the user list will be shown on the screen.

 

 

7.          Finished.

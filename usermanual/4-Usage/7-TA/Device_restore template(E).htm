Restore Template to Device

 

What is Template?

 

Template is a user enrolment record stored in Time & Attendance Device (T&A Device).

 

 

Why we need to Restore Template?

 

ecTA / ecPayroll provides a function to manage the Template record in the software. It will be safety compare with only keeping the Template record in T&A Device.

 

In case of Template record need updates (i.e. Add / Edit / Delete), you can prepare the template in the software, then restore it to the T&A Device.

 

 

TIPS

To learn more about how to prepare the Template record, please study [Add Template]

 

 

 

 

 

 

 

How to Restore Template?

 

        ecTA / ecPayroll provides 3 restore template modes, you may choose one of them in different scenario. Detail steps as below:

 

 

CAUTION

Make sure you have a complete Template List in ecTA / ecPayroll software before restoring.

 

 

 

 

 

Steps

1.          Click “Device Management” from the main screen toolbar

 

 

2.          Select the device to be restore (e.g. Site8)

 

 

3.          Click the “Restore Template” button

 

 

4.          Select the Restore Template Mode:

 

Mode

When to use

Steps

 

Full Restore

 

To transfer complete template list to the new installed device

 

Click the “Full Restore” button

 

 

 

Partial Restore

 

To add / delete some template to device

 

Click the “Partial Restore” button

 

 

 

User select Restore

 

To update some template to device by manual select

 

Manual select template record to be restore (e.g. User: 123), then click the “Restore Template” button

 

 

 

 

 

CAUTION

DO NOT shut down the software or T&A device until the restore process completed.

 

 

 

 

 

 

5.          The communication log will shows after process completed

 

 

 

TIPS

If restore template fail, please study [….] for troubleshooting

Make sure you have completed following session(s)

l   None

 

The ¡§Online News¡¨ or we call it ¡§Bulletin board¡¨ is the place for administrator to publish company news to all staffs. Administrator can edit the news content by using the system back-end program (i.e. TAMS) while the staffs read the news on the web. The news is now supported HTML format that means administrator can enter text with format, or insert some hyperlinks for linking to other useful web pages, or even insert some photos in the news directly.

 

Steps

1        Open Web Module Setup page

2        Click ¡§Online News¡¨ tab, the news content will be shown.

4            To undo changes, press [Refresh] button to reload original news content.

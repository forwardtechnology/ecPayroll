Generate Letter

Make sure you have completed following session(s)

l            Letter Template Setup

 

When user wants to generate letter for particular staff e.g. Warning Letter, Complete Probation Notification, etc. User can use a preset template to generate a Microsoft word file by ecTA / ecPayroll Letter Module Function

 

Steps

1.            Press [Letter Management] on main screen toolbar

 

2.            Press [Generate Letter(s)]

 

3.            Enter the [Date], then select a suitable [Template]

 

4.            Click [Select Employee] to select employee

 

5.            Press [ Execute] to generate letter. After the process completed, the system will shows how many letters were generated

 

6.            A new entry will display on the [Letter Management] main page. To preview the letter, double click the entry.

 

7.            To print out the letter, select the entry, then press [Print]

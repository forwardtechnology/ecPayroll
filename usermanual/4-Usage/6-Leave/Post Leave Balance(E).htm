Leave Balance Posting

Make sure you have completed following session(s)

l   None

 

 

ecPayroll calculates leave balance in real time by checking the related leave entitle plan and all leave take records therefore the leave balance is correct and up-to-date. However, after several years, the time of calculation may become longer than before due to large amount of data. Especially if you have many employees (i.e. more than 100). To speed it up, you may consider to use the function leave posting.

 

The concept of leave posting is insert a calculated leave balance adjustment record at a certain date (most likely the last day of a period) to reflect all leave records before that cut-off date. It reduces lots of calculations and therefore the whole process run faster.

 

However, there are limitations that not every employee or situation are allowed to run this process:

1.      New join staff who was employed less than 1 year

2.      The posting date must be 2 years before today.

3.      All previous leave take records before the posting date are fixed and cannot be updated.

4.      Not suitable if any expiry policy was set in the leave entitle plan.

 

The Leave Posting process seems complicated but actually the system do most for you. From user point of view, the operation is simple, please follow below steps:

 

 

Steps

1          Click the [Commander] button from the main screen toolbar

 

 

2          Select the “Leave Records Posting” command from the menu.

 

       

       

3          Select a “Leave type” to be processed.

 

 

4          Select a “Posting date”. Actually the default posting date was calculated by system that should be most suitable for most cases. If you have no idea what date should be chosen, just use the default value. Please note that all leave take records before the posting date cannot be changed anymore. Moreover, if the selected posting date is not suitable for doing post, the system will auto find the nearest suitable posting date.

 

 

5          Enter a remark to describe this posting action for reference.

 

 

6          Click the [EE Filter] button to select “Employee(s)” to be applied.

 

 

7          Click the [Execute] button to start leave posting process. All unsuitable employees such as new join staffs will be skipped processing. Then enter “Execute” to confirm your action

 

CAUTION

Make sure your selection is correct. This leave posting process CANNOT be undo.

       

       

 

8          “ Command execution completed “ message will be shown after process completed

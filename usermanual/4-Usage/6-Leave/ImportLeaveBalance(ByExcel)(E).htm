Import Leave Adjustment Records from Excel to the System

Make sure you have completed following session(s)

l            Add New Employee

l            Leave Type Setup

 

 

 

NOTE

This import function is available for the customers who have bought “HR Advance” module. If you don’t have such module, you may adjust leave balance in system backend (i.e. TAMS). For details, please refer to “Leave Day Adjustment”, or “Leave Day Adjustment (Bulk)”

 

 

Steps

1          Prepare data to be imported

1.1         Open the Excel template file. You should find the template in the folder path as shown below :

 

\\[ Your server]\TAMS\Template\Import\LeaveBalanceAdjustment.xlt

               

               

 

 

TIPS

You may copy the Excel template file to another folder where easy to open.

 

       

 

1.2         Fill in the data to be imported. Please follow the instructions shown in the table header. **The balance reference date (Excel column E) should be the end date of the yearly cut-off date, e.g. 2018/12/31 ) **

 

 

 

1.3         Save the file to a particular folder and ready for import.

 

 

 

TIPS

You may save the file with a meaningful name so that you can find it easily.

E.g. LeaveBalanceAdjustment-01-2019.xlsx

 

        

 

 

2          Import the Leave Adjustment records to the system

 

2.1         Select the “Data Import” function from the main screen “File” option


 

2.2         Select the “Leave Balance Adjustment” from the pull down menu

 

 

 

2.3         Click the [ … ] button to browse the EXCEL file prepared in step 1.3

 

 

 

2.4         Click the [ Start Import ] button to start import and wait until finished

 

 

 

NOTE

During data import, if any invalid data found, ALL data in the file will be rejected. User have to correct the mistake in the Excel file and then re-import the whole file again until success.

               

               

 

2.5         Click [No] for the normal leave balance import

 

 

2.6         “ Import completed “ message will be shown after import successfully

 

 

 

 

CAUTION

Do NOT import the same Excel file more than 1 time to prevent duplicate records

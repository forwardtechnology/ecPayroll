User Setup

Make sure you have completed following session(s)

l          User Group Setup

 

Steps

1          Click User Setup in Setup menu.

 

2          Click User Setup in Setup menu.

 

3          Creating a User

3.1         Click New icon on the User toolbar.

 

3.2         Enter User Name, password (twice, for verification) and remark in the text boxes accordingly, selecting the appropriate group for that user.

 

3.3         Click Ok to save the user record, or click Close to cancel.

 

4          Editing a User

4.1         Double click the user on the user list or select the use from list and click Edit icon on user toolbar.

4.2         User setting is displayed and you can follow the instruction shown in the pervious section.

  

5          Deleting a User

5.1         Select a user and click Delete icon. 

 

 

5.2         Click Yes to confirm.

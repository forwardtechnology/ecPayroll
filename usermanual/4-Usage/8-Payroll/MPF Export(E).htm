Exporting MPF data file

Make sure you have completed following session(s)

l   Company Setup

l   Employee Record - MPF

l   Already generated Salary Record (User manual: Add New Payroll Record  /  Fast Salary Generation )

 

In Hong Kong, companies are requested to submit MPF contribution information to their MPF Provider every month. To do that, besides of generating MPF remittance report, you can also export the MPF info to a data file in required file format. So that the data file can be submitted to the MPF provider in an electronic way. Different MPF Provider has their own required data format and submission channel. At this moment, ecPayroll System supports the following MPF Providers: (1) AIA (2) Manulife (3) HSBC (4) BEA (5) BEA Industry Scheme.

Modify Payroll Record

Make sure you have completed following session(s)

l            Add New Payroll Record

 

Steps

1.        Starting Payroll Management

 

2.        Double click the payroll record you want to edit and get the deeply information.

 

3.          Type in the Salary Date.

 

4.          Type in the Salary Period. This period means the pay range of the salary.

 

5.          Type in the Employee ID to select the payment for employee.

 

6.          Select the Salary Profile you want. Unless you want to re-select the Salary Profile, if not the system have already accorded your settings to select the appropriated profile. You need not to select yourself.

 

7.          According to all settings, the payroll item will be displayed. All the relevant variables also are calculated automatically. Finally, the total payroll will be displayed.

 

8.          Then you also can add New, Modify or Delete the payroll item. Please see Payroll Item Setup

 

9.          When you have finished your settings, you can click Update to save.

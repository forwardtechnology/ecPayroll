Import Basic Salary Records from Excel to the System

Make sure you have completed following session(s)

l   Add New Employee

 

Importing Basic Salary records from Excel is useful for batch basic salary update such as yearly basic salary adjustment.

This topic will walk you through how to batch import Basic Salary records from Excel file to the system.

 

 

 

 

NOTE

This import function is available for the customers who have bought “Payroll Advance” module. If you don’t have such module, you may input the record one by one. For details, please refer to “Update Basic Salary to Employee”

 

 

Steps

1          Prepare data to be imported

1.1         Open the Excel template file. You should find the template in the folder path as shown below :

 

\\[ Your server]\TAMS\Template\Import\BasicSalaryImport.xlt

 

 

 

TIPS

You may copy the Excel template file to another folder where easy to open.

 

               

       

 

1.2         Fill in the data to be imported. Please follow the instructions shown in the table header.

 

 

1.3         Save the file to a particular folder and ready for import.

 

 

 

TIPS

You may save the file with a meaningful name so that you can find it easily.

E.g. BasicSalaryAdjustment2018.xls

 

        

 

 

2          Import the Basic Salary records to the system

 

2.1         Select the “Data Import” function from the main screen “File” option


 

2.2         Select the “Employee Basic Salary” from the pull down menu

 

 

2.3         Click the [ … ] button to browse the EXCEL file prepared in step 1.3

 

 

2.4         Click the [ Start Import ] button to start import and wait until finished

 

 

 

NOTE

During data import, if any invalid data found, ALL data in the file will be rejected. User have to correct the mistake in the Excel file and then re-import the whole file again until success.

               

               

 

2.5         “ Import completed “ message will be shown after import successfully

 

 

 

 

CAUTION

Do NOT import the same Excel file more than 1 time to prevent duplicate records

 

 

3          View and check the imported data

3.1         The easiest way to check the Basic Salary of employees is generating the “Basic Salary Report”, it shows not only the current Basic Salary amount but also the last Basic Salary amount. Or you can check it one by one in Employee Setup > Salary > Basic Salary.

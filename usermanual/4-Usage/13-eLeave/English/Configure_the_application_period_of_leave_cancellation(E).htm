Configure the date period of applying approved leave cancellation

Make sure you have completed following session(s)

l   None

 

 

 

In e-Leave system, staffs are allowed to apply cancellation of the approved leave records. However, they are limited to apply leave cancellation in a certain date period configured by system administrator. Please follow below steps to make such configuration

 

 

 

NOTE

By default, e-Leave applicant are not allowed to cancel the approved leave record which leave taken date is on or before today.

 

 

Steps

1.          Click the [ e-Leave Setting ] button from the main screen toolbar

 

 

 

2.          Click the ¡§ Leave application¡¦ ¡¨ tab

 

 

 

3.          Enter the value to control the application period of leave cancellation.

 

E.g. Assume today is 2016/04/15 ¡K

Value

Calculation

Result

3

Today (2016/04/15) + 3 day = 2016/04/18

System only allow users to apply leave cancellation if the corresponding leave taken date is after 2016/04/18

System only allow users to apply leave cancellation if the corresponding leave taken date is after 2016/04/17

System only allow users to apply leave cancellation if the corresponding leave taken date is after 2016/04/16

0

Today (2016/04/15) + 0 day = 2016/04/15

System only allow users to apply leave cancellation if the corresponding leave taken date is after 2016/04/15

-1

Today (2016/04/15) -1 day = 2016/04/14

System only allow users to apply leave cancellation if the corresponding leave taken date is after 2016/04/14

-2

Today (2016/04/15) -2 day = 2016/04/13

System only allow users to apply leave cancellation if the corresponding leave taken date is after 2016/04/13

 

 

4.          Click the [ Save ] button to save the setting

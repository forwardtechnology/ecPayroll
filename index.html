模組式設計，讓你只買需要的功能

整個系統由多個模組組合而成，可按客戶不同需要來選擇模組，免除無謂投資，把成本減至最低。

Function List by module (Eng) 各模組詳細功能 (中文)

Excel 報表，用你熟悉的工具 

所有報表均以MS EXCEL作基礎生成，不但可以直接列印，亦可匯出至EXCEL作更多計算分析，部分報表甚至支援直接電郵至指定人員或所選員工。用戶亦可自行設計報表樣式，充分發揮MS EXCEL的強大功能。

ESS/MSS 員工自助網上平台，輕鬆管理 高效率

包括一般員及管理人員，均可透過網上自助平台進行如請假、批假、查詢考勤資料、編更、申請加班、收取糧單、修改個人資料等，從而減輕行政人員工作負擔，提升公司工作效率。

彈性而嚴密的用戶權限設定

不同用戶可設定不同權限(User access rights)，限制存取指定類型的資料，更可限制存取指定部門/職級的員工的相關資料。

資料修改追蹤

所有重要的行為及資料修改均會被記錄下來(Operation Log)，以供有需要時查核。例如，何人何時登入系統，什麼資料被誰人刪除，考勤資料修改前後分別等等

歷史資料保護

特別設計的Date Lock 可人手或按預先設定自動保護舊有資料，防止刻意或不小心更改。保護範圍包括更表、考勤記錄、薪資記錄等。

常問問題 FAQ

系統的運作要求是什麼 ?

本系統運作於MS Windows，配合MSSQL Server作資料庫，並以MS Excel作報表，網上模組則以Edge / Chrome 操作。詳細要求請按此下載 Minimum System Requirement

本公司已經安裝門禁系統，我可否保留現有系統並連結至ecPayroll系統計算考勤？

可以。請查看以下我們已經支援的牌子/型號：
- Locktronix

- ACTAtek
- Suprema

- AccuFace

若你的門禁系統能夠匯出出入記錄檔，但不符合我們標準格式，請聯絡我們並提供一個檔案樣板，我們會嘗試支援你的檔案格式。

你這個系統收費如何? 

ecPayroll包括不同軟件模組，及不同考勤門禁設備，貴公司可按需要購買相關軟、硬、配件及服務，查詢報價請：

方法一：使用我們的網上自助報價系統以即時穫取準確而有效的報價單(可多次使用)。
方法二：填妥聯絡資料，我們的銷售代表會主動與你聯絡。
方法三：Whatsapp 6353 6350 留下聯絡資料，我們的銷售代表會主動與你聯絡。
方法四：直接致電 6353 6350 與我們的銷售代表聯絡。

你這個系統是一次性收費還是要定期付款?

我們有2種收費方式，一次過買斷 或 月費計劃，各適其式。

一次過買斷：以project形式購買系統，客戶需先付訂金，然後進行系統安裝、培訓等，驗收完成後支付尾數。首年已包括系統保養服務，之後每年保養服務會另行報價，客戶可選擇買或不買，沒有保養服務不會導致系統不能運作。

月費計劃：以月費形式租用系統，客戶可按月付費或預付以穫得更大折扣。月費計劃已包括系統保養服務，無需另外付費。更多資料

可否按本公司需要新增或修改功能 ?
